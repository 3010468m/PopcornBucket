# PopcornBucket

# Users should be able to securely create and edit their account.​
#  Users must be able to log in and out of their account.​
# Site admin will have different access rights allowing them to add/ remove 
# films and cinemas.​
# Anyone should be able to view a film’s general info (title, poster, description etc) and 
# its reviews, regardless of whether they have an account or not.​
# Registered users should be able to leave a descriptive review and rating (out of 5) for 
# any film on the site they have watched.​
# Registered users should be able to add films to a personal watchlist to keep track of 
# films they want to watch.​
# Users should be able to sort films by category, e.g. by genre, newest, oldest etc.​
# Users should be able to locate cinemas showing a selected film and if desired, be 
# redirected to the cinema's official website via a stored URL for ticket booking.​
