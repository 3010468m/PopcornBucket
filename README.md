# PopcornBucket

- Users should be able to securely create and edit their account.
- Users must be able to log in and out of their account.
- Site admin will have different access rights allowing them to add/remove films and cinemas.
- Anyone should be able to view a film’s general info (title, poster, description etc) and its reviews.
- Registered users should be able to leave a descriptive review and rating (out of 5).
- Registered users should be able to add films to a personal watchlist.
- Users should be able to sort films by category (genre, newest, oldest etc).
- Users should be able to locate cinemas showing a selected film and be redirected to booking.
